# NeoDex 0.10 Changelog

## Added

- Complete German and English interface
- Automatic name display based on app and hack language
- Localized abilities and types
- Detailed stats, physical data and evolution chains
- Persistent language, search, filters and collection status
- Real interactive home screen without mockup status indicators

## Changed

- Replaced fragmented home and detail implementations with one stable core
- Improved responsive mobile layout
- Unified Pokédex and settings design

## Fixed

- English-only ability labels in German mode
- Mockup time, signal and battery appearing in the app
- Inconsistent storage and filtering behavior
