# Changelog

All notable changes to NeoDex are documented here.

## [0.1.0] — First Flight — 2026-07-11

### English

- Created the initial multi-package repository structure.
- Added English and German project documentation.
- Defined NeoDex Core and game-package architecture.
- Added the HeartGold Generations 2.0 package configuration.
- Documented localization rules for app and hack languages.
- Documented the three-level guide concept.
- Added the approved master home design and current PWA prototype.
- Established project values, decisions and roadmap.

- Konfiguration für HeartGold Generations 2.0 angelegt.
- Sprachlogik für App- und Hack-Sprache dokumentiert.
