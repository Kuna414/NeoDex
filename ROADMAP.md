# NeoDex Roadmap

## 0.1 — First Flight

- Establish repository structure
- Document project charter and architecture
- Add approved master design
- Add first HeartGold Generations package configuration
- Prepare internationalization and guide interfaces

## 0.2 — Pokédex Core

- Reliable bilingual search
- Pokémon details
- Seen, caught, shiny and favorites tracking
- Game-package overrides
- Localized interface foundation

## 0.3 — RouteDex

- Encounter locations
- Time of day
- Encounter method and rate
- Search Pokémon by route and route by Pokémon

## 0.4 — Data Quality

- Validation tooling
- Source and version metadata
- Clear distinction between standard and hack-specific data
- Offline data packages

## 0.5 — Guide Foundation

- Three help levels: hint, direction, full solution
- Spoiler-aware navigation
- Story-step data model

## 1.0 — HeartGold Generations Edition

- Complete HGG 2.0 Pokédex
- RouteDex
- ItemDex
- TrainerDex
- Stable offline PWA
- German and English UI

## 2.0 — NeoDex Platform

- Multiple game packages
- Game selection
- Shared core with package-specific branding and data
- Reusable guide system
- Optional synchronization
