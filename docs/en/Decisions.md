# Architecture and Product Decisions

- **ENT-001:** NeoDex is a platform with interchangeable game packages.
- **ENT-002:** Noctuh is the official companion.
- **ENT-003:** The approved master home design is frozen.
- **ENT-004:** Pokémon name display follows app language and hack language.
- **ENT-005:** The guide uses hint, direction and full-solution levels.
- **ENT-006:** NeoDex follows an offline-first approach.
- **ENT-007:** Quality takes priority over feature quantity.
- **ENT-008:** Standard data and ROM-hack overrides remain separate.
