# Design System

NeoDex should feel like a polished native application: calm, fast, clear and uncluttered.

The approved home-screen master design is frozen. Changes require a documented design decision.

Core rules:

- large touch targets,
- clear hierarchy,
- few primary actions per screen,
- progressive disclosure,
- consistent spacing and radii,
- dark-mode support,
- short purposeful animations,
- no advertising or intrusive prompts.

Noctuh should appear friendly, calm, attentive and wise.
